# AngularFrontEndLibraries

The Angular libraries for projects:

1. Video Library

## How to run projects
<br>

1. Go to `AngularFrontEndLibraries` folder
    ```sh
    cd .\src\AngularFrontEndLibraries\
    ```
    <br>

2.  Install dependencies

    ```sh
    npm ci
    ```
    <br>

3. Build all apps

    ```sh
    npm run all:build
    ```
    <br>

4. Run Demo

    ```sh
    npm run demo:serve
    ```
